
Mini Site com MySQL - pronto para rodar

Instruções rápidas:
1) Ajuste src/db.php ou defina variáveis de ambiente DB_HOST, DB_NAME, DB_USER, DB_PASS.
2) Execute o script sql/create_table.sql no seu banco MySQL.
3) Configure o seu servidor web (document root apontando para public/).
4) Acesse /register.php para criar a conta.

Files included:
- sql/create_table.sql
- src/db.php
- src/session.php
- public/*.php
- assets/css/custom.css
